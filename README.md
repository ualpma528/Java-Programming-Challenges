# ⚔️ Java-Programming-Challenges

Repositorio dedicado al estudio y práctica de Fundamentos de Programación y Metodología de la Programación con Java. Incluye retos de lógica, Estructuras de Datos y Programación Orientada a Objetos.

## 🚀 Objetivos del Reto
- Implementar clases y objetos.
- Gestionar arrays, métodos y programas cada vez más complejos.
- Lógica de búsqueda y manipulación de datos en memoria.

## 🛠️ Tecnologías
- **Lenguaje:** Java 17+
- **IDE:** Eclipse
